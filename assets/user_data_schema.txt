/// - [required]: Optional marker. If present, the field cannot be null when creating user data.
///
/// Rules:

/// 2. Arrays must specify the element type: array<data_type>
/// 3. When a field is moved, the system will preserve its value
/// 4. When a field is removed, it will be deleted from the database
/// 5. When a field is added, it will use the default value
